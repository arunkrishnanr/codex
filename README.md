# Codex Tetris

This repository contains a simple Tetris game implemented with Python's `curses` module. The game runs directly in the terminal and does not require any third-party dependencies.

## Running the Game

Execute the following command from the repository root:

```bash
python tetris.py
```

Use the arrow keys to move and rotate the pieces. Press `space` to drop a piece instantly and `q` to quit.
