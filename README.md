# Codex Games

This repository contains terminal-based games implemented in Python.

## Tetris

A simple Tetris game using Python's `curses` module. Run:

```bash
python tetris.py
```

Use the arrow keys to move and rotate pieces. Press `space` to drop and `q` to quit.

## 20 Questions

A dynamic 20 Questions game that learns new objects over time. Run:

```bash
python twenty_questions.py
```

The game stores knowledge in `twenty_questions.json` and updates it whenever you teach it something new.
